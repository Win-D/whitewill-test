# Вопрос

PHP обычно не позволяет с помощью оператора `extends` наследовать больше одного класса, но если необходимо наследовать методы из разных частей кода, то как это можно сделать?

# Ответ

1. Если мы собираемся переопределять эти методы, то их можно вынести в отдельный интерфейс и далее имплементировать этот интерфейс.
2. Если переопределение методов в каждом классе нам не требуется, можно вынести эти методы в один или несколько трейтов и далее вставлять эти трейты в нужные классы.
